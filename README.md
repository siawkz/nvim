# KZ neovim

## Install

### Prerequisites

- [Neovim](neovim-install) >= v0.8.0

```sh
  brew install neovim
```

- Lazygit

```sh
  brew install lazygit
```

- for telescope.nvim

```sh
  brew install ripgrep fd
```

### Recommended Fonts

JetBrainsMono Nerd Font
